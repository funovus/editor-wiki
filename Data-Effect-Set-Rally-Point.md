# [](dcei.engine.proto.Effect.set_rally_point)**[Data](Data)\\[Effect](Data-Effect)\Set Rally Point** {dataEffectSet-Rally-Point}

## [](dcei.engine.proto.EffectSetRallyPoint.which_unit)**Which Unit** {Which-Unit}

[](manual-wiki-end)
